### 开始使用

游戏开始，会问你是否需要神的援助，接受则获得一个神灯。

你的命运商店里会多出两个道具：

1. 撒旦的种子（目前没有用）
2. 事件召唤卡（用于强制触发事件）

将神灯拖入俺寻思，在家宅触发仪式，拖入对应的位置即可实现道具或者金钱的获取。每回合可以无限使用

将事件召唤卡拖入俺寻思，在神殿触发仪式，随便放入什么，根据首字母选择仪式，然后等待一回合即可触发部分仪式
